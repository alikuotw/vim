## Install step ##


#### 1. Set up vim config: ####

    git clone https://github.com/alikuotw/vim ~/.vim

#### 2. make (can choice custom or not): ####
    
    make
    make custom

#### 3. Install packages:  ####
    
    make install

#### 4. Install zsh && oh-my-zsh: ####
    
    make zsh

#### 5. Change theme: ####

    make theme

#### 6. Clean: ####

    make clean
